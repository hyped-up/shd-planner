# Common Build Mistakes the Assistant Should Catch

These are the mistakes the assistant should proactively flag when analyzing a build. Based on the principle that prevention is more valuable than cure.

## Stat Mistakes

### 1. Over-capping CHC

Most common mistake. Every point above 60% is completely wasted. The assistant should total all CHC sources (gear, watch, brand bonus, weapon mods, gear mods, Coyote's) and flag if over.

### 2. Ignoring CHD

Players stack CHC to 60% then forget to add CHD. A 60/120 split performs significantly worse than 55/170.

### 3. Running Headshot Damage on Non-Precision Builds

HSD is only valuable if you consistently land headshots. On an Assault Turret build, it is a dead stat.

### 4. Mixing Core Attributes Without Purpose

3 red / 2 blue / 1 yellow usually means the build does not know what it wants to be. Exception: intentional hybrid builds like Memento-based setups.

## Gear Mistakes

### 5. Using Exotic for the Name, Not the Talent

Coyote's Mask on a skill build is a wasted exotic slot. Waveform or Capacitor would serve the build better.

### 6. Ignoring Brand Bonuses

Running 1 piece each of 6 different brands wastes the 2pc and 3pc bonuses that add real power.

### 7. Wrong Talent for the Build

Glass Cannon on a tank build, Kinetic Momentum on a build with no skill tier, Efficient on a build that does not use armor kits.

### 8. Forgetting the Backpack Talent

Backpack talents are some of the most powerful in the game. Running a gear set backpack when a Providence backpack with Vigilance would be stronger.

## Mode Mistakes

### 9. Full Glass Cannon in Legendary Without Reviver Hive

You will die, and dead agents deal zero DPS.

### 10. PvE Build in PvP

PvP normalizes stats differently. Armor matters. Status builds work differently. Headshot multiplier is different.

### 11. Solo Build in Group Content

Memento requires kills; in a 4-player Legendary, you will not get enough trophies. Switch to a non-kill-dependent build.

## Meta Mistakes

### 12. Chasing YouTube Builds Blindly

Content creator builds are often showcased in optimal conditions (solo challenging, not Legendary). The assistant should evaluate whether the build actually works in the user's target mode.

### 13. Over-Optimizing Before Having the Gear

Do not theory-craft the perfect build if you are missing 3 pieces. Farm first, optimize later.
